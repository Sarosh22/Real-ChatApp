# 💬 Real-Time Chat Application (WebSockets + Redis Cluster)

## 📌 Overview
This project is a real-time chat application built using Socket.IO and Redis Cluster for scalable message distribution.

## 🚀 Features
- Real-time messaging
- Multiple users
- Redis-based scaling
- Online users list
- Professional UI

## 🛠️ Technologies
- Node.js
- Express
- Socket.IO
- Redis Cluster
- Docker

## ⚙️ Setup Instructions

### 1. Install dependencies
npm install

### 2. Run Redis Cluster
cd redis-cluster
docker-compose up -d

### 3. Start server
node server.js

### 4. Open in browser
http://localhost:3000

## 📊 Scalability
Supports multiple server instances using Redis adapter.
